# Freshdesk Demo Prep Assistant

A Streamlit application that transforms discovery call notes into actionable Freshdesk demo plans using Gemini AI. Paste customer notes and instantly generate a tailored demo agenda, stakeholder analysis, business goals, feature recommendations, discovery gaps, objection handling points, and a follow-up email. Built with Python, Streamlit, and Gemini 2.5 Flash to help Solutions Engineers deliver more relevant and impactful demos.
